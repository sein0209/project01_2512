# SSU 훈련 관리 앱 (SSU Training Assistant)

## 📌 프로젝트 목적
SSU(해난구조전대)를 목표로 하는 훈련생이 수영·체력 훈련을 체계적으로 기록하고 관리할 수 있도록 돕는 앱입니다.
SSU 체력 기준을 의식한 훈련을 통해 목표 달성까지의 과정을 스스로 점검할 수 있도록 만들었습니다.

## 🧠 사용 기술
- Python
- OpenAI API
- Streamlit
- Git & GitHub

## ⚙ 구현 내용
- 사용자의 훈련 기록 입력 (수영, 러닝, 근력)
- 일일/주간 훈련 로그 저장
- 목표 설정 및 달성도 확인
- AI를 활용한 훈련 피드백 제공
- 웹 화면으로 결과 출력

## ✍ 느낀 점
이 프로젝트를 통해 단순한 운동 기록이 아니라,
명확한 목표(SSU)를 가진 앱 기획과 구현의 중요성을 느꼈습니다.
또한 AI API를 활용하면 복잡한 모델을 직접 만들지 않아도
실용적인 서비스를 구현할 수 있다는 것을 알게 되었습니다.

SSU를 준비하는 과정에서 실제로 활용할 수 있는 앱을 만들었다는 점에서
가장 의미 있는 프로젝트였습니다.
